# UPath - 升学机构管理平台

A comprehensive platform for college admissions consulting agencies, helping high school students prepare for university applications.

## Features
- 🔍 University Search with QS & US News Rankings
- ✍️ Personal Statement Workspace
- 📅 Application Deadline Calendar
- 📁 File Drive
- 📝 Test Practice Database (SAT, IELTS, TOEFL, ACT)
- 👥 Role-based System (Admin, Teacher, Student)
- 📚 Assignment & Coursework System
- 📊 Analytics Dashboard
- 🎓 Scholarship Database
- 🤖 AI Personal Statement Assistant

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Prisma ORM
- NextAuth.js
- SQLite/PostgreSQL

## Getting Started
```bash
npm install
npm run db:generate
npm run db:push
npm run db:seed
npm run dev
```
